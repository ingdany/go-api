# go-api
API in GO Language
## Install GO
1. Download from https://golang.org/dl/
2. Install
3. To verify installation
`go version`
## Execute
1. To run it in bash
`$ go run hello.go`
2. To create executable
`$ go build hello`
## To install github packages
`$ go get -u github.com/gorilla/mux `
## Connect to mysql database
`$ mysql -u root -p`
